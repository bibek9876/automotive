# Sarav Motors Auto Repair Center

Single-page automotive service website built with Next.js App Router, Tailwind CSS v4, Framer Motion, and EmailJS.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## EmailJS Setup

Update the placeholders in `src/components/QuoteForm.jsx`:

- `YOUR_TEMPLATE_ID`
- `YOUR_PUBLIC_KEY`


## Build

```bash
npm run lint
npm run build
```
